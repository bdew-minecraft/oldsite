Note about Advanced Generators

I have somewhat worked myself into a corner with mutually incompatible dependencies. I have to release a new version of Gendustry that works with Forestry 3.4 and that codebase depends on BdLib 1.6.0. The problem is that the version of Advanced Generators that works with the latest BdLib is nowhere near ready for release, and the last stable version will not work.

As such, there is currently no stable release of Advanced Generators that can co-exist with Gendustry and Forestry 3.4.

Packs that include all the mods in question can either wait for a few days until i wrap up the next version of Advanced Generators or use the version from Jenkins (not recommended).
